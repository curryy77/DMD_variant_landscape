# DMD変異の全体像の分析

![License](https://img.shields.io/badge/license-MIT-green)
![Python](https://img.shields.io/badge/python-3.10%2B-blue)
![Bioinformatics](https://img.shields.io/badge/field-bioinformatics-orange)
![Data](https://img.shields.io/badge/data-ClinVar%20%7C%20gnomAD%20%7C%20Ensembl-purple)

---
### 概要

アノテション付き変異データセットを用いた、**DMD遺伝子**における病原性変異および非病原性変異の探索的解析。その目的は、**エクソン**、**タンパク質ドメイン**、および**ORFのリーディングフレームの状態**にわたる病原性の構造的パータンと機能的パータンを調査することであります。

---

### データセット

この解析は、ゲノム位置、エクソン番号、タンパク質ドメイン、病原性分類、表現型クラス（DMDまたはBMD)、リーディングフレームの状態（フレーム内またはフレーム外）を含む、**DMD変異**の注釈付きデータセット（**ClinVar**、**Ensembl**、および**gnomAD**データベースから統合されたもの）を使用しています。変異は前処理され、以下のアドレスに保存されています：
```commandline
data/processed/DMD_variants_annotated.csv
```
---
### (これまでに）実施された分析

#### 1. 変異のエクソンレベルでの分布

まず、**DMD遺伝子の全79エクソン**にわたる病原性変異および非病原性変異の分布を分析しました。

![Exon distribution](figures/0XX_exon_distribution.png)

２つの補完的な可視化結果が生成されました：**エクソンごとの変異数**、**エクソンごとの病原性変異の割合**です。

観察されるのは、エクソン42〜56に変異が著しく集中していることであり、これは複数の研究（Monaco et al., 1988; Aartsma-Rus et al., 2006; Bladen et al., 2005)で報告されているDMDの欠失ホットスポット領域に対応しています。

この領域は、複数のスペクトリン様反復配列を含むジストロフィンの中央ロッドドメインと重なり合っています。

#### 2.ドメインレベルでの病原性

変異は、ジストロフィンの機能的な**タンパク質ドメイン**にマッピングされた（マッピングはUniProtによる）。名ドメインについて、変異のそう数および病原性変異の割合を算出しました。

![Domain distribution](figures/1XX_domain_distribution.png)

病原性割合が最も高いドメインには、**アクチン結合ドメイン**、**SNTB1結合領域**、**シントロフィンの相互作用**などが含まれます。これらのドメインは、ジストロフィン・糖タンパク質複合体におけるジストロフィンの構造的役割に不可欠な**重要な相互作用界面**に対応しています。これらの領域における変異は、タンパク質の機能をより頻繁に阻害すると予想され、より高い病原性を示します　(Ervasti & Campbell, 1993; Koenig et al., 1988; Blake et al., 2002)。

#### 3. リーディングフレーム規則の分析

(Monaco et al., 1988) で記述されました、ジストロフィン病における古典的なリーディングフレームの規則について検証が行われた。この規則によれば、フレーム外変異は重症のデュシェンヌ型筋ジストロフィー（DMD) を引き起こす傾向がある一方、フレーム内変異はより軽症のベッカー型筋ジストロフィー（BMD) を引き起こすという。

![Reading frame rule](figures/2XX_reading_frame_rule.png)

表現方およびフレーム状態ごとの変異数と、表現型の割合を示す割合プロットが生成されました。

結果は予想通りの傾向を示しています。すなわち、フレーム外変異はDMDに強く偏在している一方、フレーム内変異はBMDの割合が高い。しかし、例外も観察されます。こうした例外は、重要な機能ドメインの破壊、オルタナティブスプライシングの影響、および切断されたジストロフィンの構造的不安定性によって生じている可能性があります。

---
### 生物学的解釈

これらの分析により、DMD変異の全体像に関するいくつかの重要な特性が明らかになった。すなわち、**変異のホットスポットは遺伝子の構造的に反復する領域と一致している**こと、**機能的な結合ドメイン**では**病原性率が上昇している**こと、そして**モナコのリーディングフレームの規則**は**概ね支持される**ものの、絶対的なものではないことが示された。

### データのソース

この分析では、複数の公開データベースから得られた変異情報を統合しています：

- **ClinVar** - 臨床的変異の注釈
- **gnomAD** - 集団における対立遺伝子頻度
- **Ensembl** - ゲノム注釈および転写産物の構造
- **GEO (GSE38417)** - 遺伝子発現データセット

上記のデータベースの管理者の方々が、ゲノムデータおよび臨床データを公開してくださったことに感謝いたします。
